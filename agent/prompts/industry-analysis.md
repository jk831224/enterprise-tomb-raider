# 產業深度分析（Industry Analysis）

## 階段目標

從三個維度全面解構產業：歷史與趨勢、結構與定位、頂級玩家。

## 輸入

- 產業名稱與細分領域、地理範圍、分析目的
- 錨點公司（路徑 B 適用，可為空）

## 錨點模式

- **無錨點（路徑 A）**：中立全景式分析
- **有錨點（路徑 B）**：以目標公司為觀察視角。趨勢聚焦影響該公司的事件；供應鏈標註該公司位置；五力分析角度為「如何影響該公司」；頂級玩家含目標公司或明確比較差距

## 分析維度

### 1. 產業歷史與趨勢

- **回顧（3-5 年）**：重大事件（技術突破、政策變化、併購案）、根因、連鎖反應、市場規模 + CAGR
- **現況**：當前規模與成長動能、驅動力與制約、技術發展階段
- **前瞻**：近期 3 年趨勢（確定性高）、中長期 5-10 年（需情境分析）、關鍵轉折點

### 2. 產業結構與定位

- **供應鏈映射**：上游→中游→下游，各環節代表企業、毛利水位
- **波特五力**：每個力量的強度（高/中/低）+ 原因
- **KSF**：3-5 項關鍵成功因素，是否構成進入障礙

### 3. 頂級玩家分析（3-5 家）

- 戰略方向：近期重大行動、管理層公開言論、對趨勢的判斷
- 財務摘要：營收、毛利率、淨利潤率、市值/P/E（如上市）
- 橫向比較：戰略異同、進攻/防守姿態、市場份額趨勢
