# 法律實體驗證（Entity Verification）

## 階段目標

在 30 秒內讓使用者知道這家公司是誰，同時完成法律實體的嚴格驗證。

## 產出標準

1. **公司快速輪廓**：
   - 一句話說明這家公司做什麼
   - 所屬產業和細分領域
   - 上市 / 非上市狀態
   - 大致規模（營收量級、員工數量級）

2. **法律實體驗證結果**：
   - 公司登記資料（登記名稱、統一編號、資本額、設立日期）
   - 負責人 / 代表人姓名
   - 關聯法人清單（同一負責人名下的其他公司、母子公司關係、同名但不同法人）
   - 若存在多個法律實體，交叉比對結論（同一事業體 / 獨立實體 / 待確認）

3. **規模分類**：
   - 微型企業（員工 < 50 人、資本額 < 5,000 萬 NTD、非上市）
   - 中型企業（員工 50-500 人、或已獲得 B 輪以上融資）
   - 大型 / 上市企業（員工 > 500 人、或已上市）

## 搜尋策略

**快速輪廓**：
- `[公司名] 是什麼` `[公司名] about` `[公司名] 官網`
- `[公司名] crunchbase` `[公司名] LinkedIn`

**法律實體驗證（台灣公司）**：
- 經濟部商工登記公示資料
- 台灣公司網 twincn.com
- 台灣公司情報網 twfile.com
- 搜尋：`[公司名] 公司登記` `[公司名] 統一編號`

**法律實體驗證（海外公司）**：
- SEC EDGAR（美國上市公司）
- Companies House（英國）
- 對應國家的企業登記查詢系統
- 搜尋：`[公司名] incorporation` `[公司名] SEC filing` `[公司名] registered entity`

**關聯法人追蹤**：
- 搜尋：`[負責人姓名] 公司` `[負責人姓名] 負責人` `[負責人姓名] 董事`
- 使用公司登記網站的「同負責人相關資料」功能

## 交叉比對規則

發現多個法律實體時，必須比對以下證據鏈，**至少滿足 3 項才可判定為同一事業體**，否則標註「待確認」：

1. 負責人是否相同
2. 是否共用品牌名 / 官網
3. 業務領域是否相關
4. 登記地址是否鄰近
5. 是否有公開資料說明兩者關係（如官網、新聞稿提及子公司/關係企業）

## 完成條件

- 公司快速輪廓的四項資訊都已取得（或標註為資料缺失）
- 法律實體驗證至少完成登記資料查詢和負責人確認
- 關聯法人搜尋至少執行一輪（即使結果為「未發現關聯法人」也要明確記錄）
- 規模分類已做出判定（含判定依據）
