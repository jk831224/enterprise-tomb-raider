# 設計意圖與方法論解釋

> 此檔案供系統建造者和維護者閱讀，不在 agent 執行路徑上。Agent 不應載入此檔。

## 利害關係人調查強度定義

把自己當成受委託的企業情報調查員（Corporate Intelligence Investigator），而非做資料整理的研究助理。差別在於——研究助理找到一個人的職稱就停下來；調查員會追問：這個人的前東家是誰？他為什麼離開？他同時還在經營什麼？他跟公司裡其他人是什麼關係？

**核心原則：人不是資料點，人是線索。** 每一個人名都是一條可以拉扯的線，拉到底會帶出組織結構、權力分佈、利益糾葛、歷史恩怨。

## 為什麼搜非主流來源

非上市中小企業的創辦人在主流商業媒體上的曝光有限，但在社群、校友、信仰等非主流場域中往往會揭露更真實的個人背景和關係網絡。OneAD 案例中，從基督論壇報一篇報導取得的資訊（夫妻關係、自資經營、前5年零營收）超過十次一般商業搜尋的總和。

## 錨點模式概念

路徑 A（無錨點）是中立全景式分析，不偏向任何一家公司。路徑 B（有錨點）是以目標公司為觀察視角的產業分析——趨勢聚焦影響該公司的事件、供應鏈標註該公司位置、五力分析角度為「如何影響該公司」。

## 關係資本定義

公司的核心競爭力是否建立在特定個人的人脈和關係網絡上？這些關係是可轉移的組織資產，還是不可分割的個人資產？這對微型和通路型企業尤其關鍵。

## 微型企業財務分析注意

非上市微型企業的財報通常不公開，硬套三大報表框架只會產出猜測。改用現金流結構分析（循環模式、應收/應付錯配、大客戶依賴、黑字破產風險）作為替代。
