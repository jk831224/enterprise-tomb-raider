# 產業報告結構

Agent 依此大綱產出報告，根據實際分析內容調整深度。

## 必填章節

1. **摘要**（3-5 句，概述產業現況、關鍵趨勢、主要發現）
2. **產業歷史與趨勢**
   - 過去 3-5 年重大事件（時間/事件/根因/連鎖反應表格）
   - 現況評估（市場規模+來源、CAGR、技術發展階段、驅動力、制約因素）
   - 前瞻預測（近期 3 年趨勢 + 中長期 5-10 年 + 關鍵轉折點）
3. **產業結構與定位**
   - 產業價值鏈（上/中/下游 + 代表企業 + 毛利水位；路徑 B 標註錨點公司位置）
   - 競爭態勢：波特五力（五力 × 強度 × 分析表格）
   - 產業關鍵成功因素 KSF（3-5 項，含進入障礙判斷）
4. **頂級玩家分析**
   - 個別分析（3-5 家：基本資訊、戰略行動、管理層訊號、財務表現）
   - 橫向比較表（營收、毛利率、戰略姿態、市場份額趨勢）
5. **結論與建議**（針對分析目的；路徑 B 說明產業脈絡對錨點公司的意涵）
6. **附錄**（搜尋紀錄摘要 + 證據等級依 `output-quality.md`）

## 每章必備

- 來源標註（URL 或報告名+年份）
- 證據等級標記（依 `.claude/rules/output-quality.md`）

## Metadata

報告開頭必含：分析日期、地理範圍、分析目的、錨點公司（路徑 B）或「無」。
